# Networks

## Public Available Endpoints

Below is a list of publicly available endpoint that you can use to connect to the Qubetics mainnet and
public testnet:


### Mainnet
| Address                                                      | Category               | 
| ---------------------------------------------                | ---------------------- | 
| `https://tendermint.qubetics.com`                            | `Tendermint` `RPC`     | 
| `https://swagger.qubetics.com/swagger/#`                     | `Swagger`              | 
| `https://rpc.qubetics.com`                                   | `Ethereum` `JSON-RPC`  | 
| `wss://socket.qubetics.com`                                  | `Ethereum` `WSS`       | 
| `https://grpc.qubetics.com`                                  | `Cosmos` `GRPC`        | 
| `wss://tendermint.qubetics.com/websocket`                    | `Tendermint` `WSS`     |

### Testnet

| Address                                                      | Category               | 
| ---------------------------------------------                | ---------------------- | 
| `https://rpc-testnet.qubetics.work`                          | `Ethereum` `JSON-RPC`  | 
| `wss://socket-testnet.qubetics.work`                         | `Ethereum` `WSS`       |
| `https://swagger-testnet.qubetics.work/swagger/#`            | `Swagger`              | 
| `https://tendermint-testnet.qubetics.work`                   | `Tendermint` `RPC`     | 
| `wss://tendermint-testnet.qubetics.work/websocket`           | `Tendermint` `WSS`     |
